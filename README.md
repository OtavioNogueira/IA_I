# Atividades de Inteligência Artificial - Sistemas de Informação

Este repositório contém as atividades práticas desenvolvidas para a matéria de **Inteligência Artificial (IA)** do curso de **Sistemas de Informação**. 

O objetivo do projeto é implementar e analisar algoritmos clássicos de Redes Neurais Artificiais (RNAs), compreendendo desde modelos de camada única até arquiteturas mais robustas de múltiplas camadas e funções de base radial.

---

## 📌 Conteúdos do Repositório

O projeto está estruturado em tópicos evolutivos de aprendizado de máquina:

*   **[Perceptron](file:///c:/Users/Ot%C3%A1vio%20Nogueira/Desktop/IA_I/perceptron)**
    *   Implementação clássica do classificador linear de camada única utilizando a Regra de Aprendizagem de Hebb para separação de classes de pureza de óleo.
*   **[Adaline](file:///c:/Users/Ot%C3%A1vio%20Nogueira/Desktop/IA_I/adaline)**
    *   Rede neural de camada única com algoritmo de aprendizado baseado na regra Delta (LMS - Least Mean Squares) voltada para convergência por minimização do erro quadrático médio.
*   **[Perceptron Multicamadas (PMC / MLP)](file:///c:/Users/Ot%C3%A1vio%20Nogueira/Desktop/IA_I/PMC)**
    *   Redes neurais multicamadas treinadas com o algoritmo *Backpropagation* para resolver problemas de classificação e aproximação não-lineares.
*   **[Redes de Função de Base Radial (RBF)](file:///c:/Users/Ot%C3%A1vio%20Nogueira/Desktop/IA_I/RBF)**
    *   Rede neural RBF aplicada em problemas complexos de interpolação e classificação não-linear utilizando funções gaussianas como ativação na camada oculta.
*   **[Rede de Hopfield](file:///c:/Users/Ot%C3%A1vio%20Nogueira/Desktop/IA_I/Hopfield)**
    *   Rede neural recorrente de camada única configurada como memória associativa para armazenar e recuperar padrões de dígitos ruidosos de 45 bits.
*   **[Mapas Auto-Organizáveis de Kohonen (SOM)](file:///c:/Users/Ot%C3%A1vio%20Nogueira/Desktop/IA_I/Kohonen)**
    *   Rede neural de aprendizado não supervisionado estruturada em um grid topológico bidimensional de $4 \times 4$ para agrupamento e análise de similaridades em dados de imperfeições de pneus.
